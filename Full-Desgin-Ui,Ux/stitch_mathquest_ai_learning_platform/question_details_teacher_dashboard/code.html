First Derivative Calculation

Find the first derivative of the function \( f(x) = \sin(x^2) + e^{-x} \) with respect to \( x \). Evaluate the derivative at \( x = \sqrt{\pi} \).

Figure 1.1: Functional visualization for conceptual reference

Solution Steps

1

Apply the Chain Rule to \( \sin(x^2) \): \( \frac{d}{dx}\sin(x^2) = 2x \cos(x^2) \)

2

Apply the Power/Exp Rule to \( e^{-x} \): \( \frac{d}{dx}e^{-x} = -e^{-x} \)
