# User Personas — MathQuest

**Version:** 1.0.0

---

## Persona 1 — The Struggling Student

**Name:** Alex, Age 12, Grade 7
**Role:** Student

### Goal
Improve mathematics grades without feeling embarrassed about being behind peers.

### Pain Points
- Finds traditional homework repetitive and demotivating
- Afraid to ask questions in class when confused
- Loses focus quickly on static worksheets
- Does not know which topics to study first

### Context
Uses a school-issued tablet at school and a personal phone at home. Has a 30-minute daily study window. Responds well to visual progress indicators and short reward loops.

### Success Criteria
- Completes at least one lesson per day
- Sees measurable improvement in weak topic areas within 4 weeks
- Feels confident enough to attempt multiplayer matches

### Key Flows
- Login → Dashboard → AI-recommended lesson → Practice → Earn achievement → Match invitation

## Persona 2 — The Competitive Student

**Name:** Jordan, Age 15, Grade 10
**Role:** Student

### Goal
Rank at the top of the class leaderboard and win tournaments.

### Pain Points
- Existing platforms do not offer real competition against peers
- Wants to see how they compare to classmates
- Gets bored with solo practice after a short time

### Context
Uses a personal laptop. Spends 45–60 minutes per day on the platform. Highly motivated by rankings and visible achievement. Needs challenge to stay engaged.

### Success Criteria
- Participates in at least 3 matches per week
- Achieves a top-10 class ranking within one month
- Completes all missions in the current week

### Key Flows
- Login → Leaderboard → Challenge opponent → Match → Result → Rank update → Mission progress

## Persona 3 — The Dedicated Teacher

**Name:** Ms. Rivera, Age 34, Mathematics Teacher, Grade 8
**Role:** Teacher

### Goal
Spend less time creating questions and more time teaching. Know exactly which students need help before the next class.

### Pain Points
- Spends 3–4 hours per week creating and organizing practice questions
- Has no real-time visibility into which students are struggling
- Existing tools require manual data export to generate reports
- AI tools she has tried produce low-quality or curriculum-misaligned questions

### Context
Uses a school desktop and personal laptop. Manages 3 classes of 28 students each. Needs a dashboard she can check in under 5 minutes before class.

### Success Criteria
- Generates a week's worth of practice questions in under 30 minutes using AI
- Identifies the 5 students most at risk before each class
- Assigns targeted practice to struggling students in 2 clicks

### Key Flows
- Login → Teacher Dashboard → Class overview → At-risk student list → Assign practice → AI question generation → Review and publish

## Persona 4 — The Overloaded Administrator

**Name:** Mr. Chen, Age 48, School Principal
**Role:** Administrator

### Goal
Demonstrate curriculum compliance to the district and identify school-wide mathematics performance trends without manual data collection.

### Pain Points
- Receives performance data only at end-of-term assessments
- Cannot identify curriculum gaps until it is too late to intervene
- Managing user accounts across 600 students and 20 teachers is time-consuming
- Needs audit-ready reports for district review

### Context
Uses a school desktop. Checks the platform weekly. Needs high-level summaries, not granular student data. Requires exportable reports.

### Success Criteria
- Views school-wide performance summary in under 2 minutes
- Exports a curriculum compliance report in one click
- Manages user accounts (add, deactivate, reset) without IT support

### Key Flows
- Login → Admin Dashboard → School performance overview → Grade-level breakdown → Export report → User management

---

## Persona 5 — The New Teacher

**Name:** Mr. Okafor, Age 26, First-Year Mathematics Teacher, Grade 5
**Role:** Teacher

### Goal
Build a question bank quickly and understand how to use the platform without extensive training.

### Pain Points
- No existing question library to start from
- Unsure how to align questions to curriculum objectives
- Limited time for platform onboarding during the school year

### Context
Uses a personal laptop. Needs guided onboarding. Relies heavily on AI generation to bootstrap his question bank. Wants templates and examples.

### Success Criteria
- Creates a full unit question bank in his first week using AI import and generation
- Understands Bloom's taxonomy tagging after one onboarding session
- Runs his first class assignment within 3 days of account creation

### Key Flows
- Onboarding wizard → Import existing questions from CSV → AI generation → Review and publish → Create first assignment
